Annonce importante

La collecte des statistiques a été significativement améliorée. Le type de réseau (2G/3G/femto) est maintenant pris en compte dans les statistiques nationales. Vous pouvez consulter ces nouvelles statistiques sur le nouveau site de Free Mobile Netstat à l'adresse suivante : fm.netstat.fr. De plus, le nouveau site conserve désormais un historique des statistiques, ce qui vous permettra de consulter les statistiques sur une plage donnée.
