Classic Monitor (versión 2.4.5)

Una aplicación de código abierto desarrollado por Graham Ross.

Classic Monitor es un monitor de estado para Midnite Solar, Classic 150, 200, 250 controlador de carga (www.midniteSolar.com).

Es un programa de Sólo Lectura, no escribe para el Clásico.
El software se proporciona "TAL CUAL" SIN GARANTÍA DE NINGÚN TIPO, expresa o tácita.
Monitor clásico no es un producto de Midnite Solar, ni el apoyo a esta aplicación!

Si la aplicación tiene Whizbang Júnior Detecta monitor actual, el estado de carga Mostrará junto con un medidor de corriente bidireccional.
Apoyo Básico para el controlador de carga Tristar MPPT de Morningstar est incluido.
Los registros de energía registrado diarios pueden ser enviados a pvoutput.org mediante el suministro de una clave de API y SID# en la configuración para cada controlador.
